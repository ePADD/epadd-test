CROSSWORD SOLVER PUZZLE PACKS

Thank you for purchasing a puzzle pack to be used with Crossword Solver. 

To play the puzzles, first download and install the version of Crossword Solver for your computer from the following site:

http://www.crosswordsolver.info/

Then copy the unzipped folder containing your puzzles that you have downloaded to your Crossword Solver folder. Run the program and select file open. Navigate to your puzzle directory, and select a puzzle to play.
